Style guide(guia de estilo)

Os Style guide são como documentos, ou paginas web, que define as regras visuais de um site, sendo importante para posibilitar um padrão em todos os elementos ou paginas de um site

Uns dos elementos que costumam compor o Style guide é:

  • Palheta de cores; define as cores presentes como a cor da letra, a cor oficial e etc
  • Tipografia; define a fonte usada, o tamanho do titulo e o espaçamentoentre as linhas
  • Regras de espaçamento; define o espaço entre os elementos
