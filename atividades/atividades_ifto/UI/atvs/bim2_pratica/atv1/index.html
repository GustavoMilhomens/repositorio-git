Objetivo Criar uma identidade visual completa para um site, aplicando conceitos de design e garantindo consistência entre os elementos visuais e o público-alvo. Descrição da Atividade Em grupos, vocês deverão desenvolver a identidade visual de um site fictício (ou baseado em uma ideia real). Essa identidade servirá como base para a futura criação do layout no Figma. Elementos Obrigatórios A identidade visual deve conter: Nome do Projeto Criar um nome original e coerente com a proposta. Ideia do Projeto Explicar o objetivo do site. O que ele oferece? Qual problema resolve? Logo Criar um logotipo simples e representativo. Deve funcionar bem visualmente (legível e coerente com a proposta). Slogan Uma frase curta que represente a marca. Público-Alvo (explicado) Definir claramente quem são os usuários do site. Explicar: idade, interesses, necessidades e comportamento. Paleta de Cores (explicada) Apresentar de 3 a 5 cores. Explicar o motivo de cada escolha (emoção, identidade, público). Tipografia (explicada) Escolher 1 ou 2 fontes. Justificar a escolha com base em legibilidade, estilo e público. Como Entregar Criar uma apresentação (estilo slides). Organizar cada item em páginas/seções bem definidas. Usar títulos, textos curtos e elementos visuais. Apresentação Na próxima aula, cada grupo deverá apresentar sua identidade visual para a turma. Durante a apresentação, devem explicar: A ideia do projeto As escolhas visuais (cores, fontes, logo) A relação com o público-alvo Sugestões de Ferramentas Para ajudar na criação da identidade visual, vocês podem usar: Paletas de Cores Coolors Adobe Color Color Hunt Dica: Busquem combinações que transmitam a emoção do projeto (ex: tecnologia = cores frias, alimentação = cores quentes/naturais).
