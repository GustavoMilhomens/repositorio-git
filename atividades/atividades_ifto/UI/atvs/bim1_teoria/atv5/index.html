Atividade 05 - Pesquisa sobre ferramentas para estabelecer consistência e padronização em UI

Nesta atividade, será mostrado uma pesquisa aprofundada sobre três conceitos fundamentais no desenvolvimento de interfaces e sistemas digitais: Framework, Guia de Estilo (Style Guide) e Atomic Design

O sistema será dividido em:

  1. Framework
    • Defina o que é um framework.
    • Explique sua função no desenvolvimento de software.
    • Apresente exemplos populares e suas principais características.
  2. Guia de Estilo (Style Guide)
    • Conceitue o que é um guia de estilo.
    • Descreva sua importância para a consistência visual e de comunicação.
    • Cite exemplos de elementos que costumam compor um guia de estilo.
  3. Atomic Design
    • Explique o conceito de Atomic Design.
    • Descreva seus níveis (átomos, moléculas, organismos, templates e páginas).
    • Comente como essa metodologia auxilia na construção de interfaces escaláveis.
