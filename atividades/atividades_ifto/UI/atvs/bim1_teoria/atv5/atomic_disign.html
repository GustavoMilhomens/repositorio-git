Atomic Desing

O Atomic Desing é uma metodologiaque se baseia na quimica para a criação de interfaces, auxiliando na construção de interfaces escaláveis, posibilitando

O atomic design é dividido em 5 niveis:

  1. Átomos: Elementos basicos como botões, campos de texto
  2. Moleculas: Grupos de átomos que funcionam juntas como os butões de uma calculadora ou uma barra de pesquisa
  3. Organismos: Grupos de moleculas que formam uma seção completa, como o cabeçario ou o rodapé
  4. Template: Seria o layout da pagina
  5. Páginas: Seria todos os elementos anteriores juntos
